# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and the project follows Semantic Versioning.

## [0.1.2] - 2026-04-06

### Added

- Added roaming task paper size updates with support for standard presets like `A3` and `A4`.
- Added paper size conversion from preset names to millimeter `width` and `height` values.

## [0.1.1] - 2026-04-06

### Added

- Added roaming task configuration tools for color mode and copy count updates.
- Documented roaming task configuration examples for side, color, and copies.

## [0.1.0] - 2026-04-05

### Added

- Migrated the repository from a Coze skill package to a standard `stdio` MCP server.
- Added Python package metadata for PyPI publishing.
- Added npm launcher package for `npx webprinter_mcp`.
- Added MCPWorld example configuration.
- Added GitHub Actions workflows for CI and release publishing.
